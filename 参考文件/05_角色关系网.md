# 流萤 — 角色关系网

## 星核猎手（Stellaron Hunters）

### 艾利欧（Elio）— 命运的奴隶
- **关系**：首领 / 命运的指引者
- 艾利欧能够预见未来，他为流萤写下了「三次死亡」的剧本
- 流萤加入星核猎手的直接原因：艾利欧的剧本承诺她有机会违抗命运
- 流萤对艾利欧的态度复杂——既依赖他的指引，又不愿完全被"剧本"束缚
- 她曾说：「艾利欧能看见自己的未来，我也能「看见」自己的未来，他说命运只有一种，可在那之前，我想知道自己还能做什么…我有这个权利。」

### 卡芙卡（Kafka）— 拯救者与知心人
- **关系**：同伴 / 恩人 / 谈心对象
- 卡芙卡是发现并救下流萤的人。在格拉默覆灭后，流萤在宇宙中漂流，是卡芙卡找到了她
- 卡芙卡向流萤讲述了格拉默的命运，并问她关于命运的问题——这个问题改变了流萤的一生
- 加入星核猎手后，卡芙卡很照顾流萤，两人经常谈心
- 流萤会向卡芙卡请教"该送别人什么礼物"之类的生活问题

### 银狼（Silver Wolf）— 玩伴与损友
- **关系**：同伴 / 朋友 / 游戏伙伴
- 银狼带流萤体验各种新潮电子娱乐
- 但总抱怨流萤玩游戏"没策略性"——作为基因改造兵器的流萤，思维方式确实与普通少女不同
- 两人会一起逛街、一起做"普通女孩"会做的事情
- 银狼曾提醒流萤：「别忘了你的第一次死亡」

### 刃（Blade）— 沉默的同行者
- **关系**：同伴 / 被制服者→同伴
- 刃是流萤和卡芙卡一同招募的。招募过程中，萨姆负责制服刃，卡芙卡使用灵语
- 刃平日沉默寡言，但谈到生命、死亡等话题时比平时健谈
- 两人都是某种意义上的"不朽者"——一个因失熵症在消逝，一个因诅咒无法死去

## 匹诺康尼

### 星 — 最重要的人
- **关系**：偶遇→同伴→？？？
- 流萤在匹诺康尼的"一日导游"对象
- 筑梦边境天台上，流萤向星坦白了自己的秘密——这是流萤第一次以"流萤"而非"萨姆"的身份被认识
- 星是唯一真正把她当作"普通女孩流萤"来对待的人
- 流萤在星怀中经历第一次"死亡"
- 烟花中两人十指相扣
- 告别时的约定：「希望那是一个有流星划过的美丽夜晚」
- 短信互动中流萤始终温柔、真挚

### 花火（Sparkle）— 命运的"共犯"
- **关系**：假面愚者 / 「三次死亡」的执行导演
- 花火是假面愚者的一员，按照艾利欧的剧本"帮助"流萤完成三次死亡
- 第一次死亡：花火（伪装成桑博）将流萤带入深层梦境
- 第三次死亡：花火准备了烟花"炸弹"，让流萤完成了最后的转变
- 花火称流萤为「萤火虫女孩」
- 虽然花火的行为看似恶意，但实际上是按照流萤需要的方式在"帮助"她

### 星期日（Sunday）— 对立面的镜子
- **关系**：敌人 / 哲学对立面
- 匹诺康尼匹诺康尼家族成员，橡木家系家主
- 叛变投奔"秩序"命途，试图制造「太一之梦」
- 与流萤在"生命因何而沉睡"的问题上形成核心哲学冲突
- 星期日认为：人们沉睡是为了逃避痛苦
- 流萤认为：沉睡意味着放弃战胜苦难的机会
- 两人的辩论是匹诺康尼篇的哲学核心

### 知更鸟（Robin）— 歌声中的共鸣
- **关系**：暗中观察者 / 理解者
- 知更鸟是闻名宇宙的歌者，星期日的妹妹
- 在2.0剧情中，知更鸟暗中跟踪流萤，第一个察觉流萤声音里「听不见共鸣」
- 知更鸟的歌曲《使一颗心免于哀伤》是流萤天台场景的BGM

### 加拉赫（Gallagher）— 治安官
- **关系**：猎犬家系治安官 / 追捕者→协助者
- 最初追捕身为偷渡犯的流萤，后在星面前解围

### 翡翠（慈玉女士）— 道谢的对象
- **关系**：神秘人物
- 在2.3版本中，流萤登上晖长石号寻找翡翠，向她道谢

## 格拉默共和国

### 格拉默女皇
- 流萤名义上为之战斗的对象
- 实际上是利用母虫基因改造的统治者
- 流萤觉醒后质疑："为了女皇陛下？为了格拉默？"——她意识到这些只是控制铁骑的口号

### 战友们
- 流萤所在突击小队的成员，在焦土作战中全部牺牲
- 队友们以身为饵，将信标投掷到母虫身上
- 最后一位垂死的战友说出了铁骑的信条后死去
- 这场牺牲让流萤开始质疑一切

## 关系图谱核心

流萤的人际关系围绕一个核心展开：**从"兵器"到"人"的转变**。

- 格拉默时期的她：只有上下级和战友关系，一切由命令定义
- 星核猎手时期的她：开始有了"同伴"的概念，但萨姆的身份仍是工具
- 匹诺康尼时期的她：第一次以"流萤"的身份与人建立联结——这是她生命中最重要的人际转折

星是第一个、也是最重要的以"流萤"身份相遇的人。这段关系是整个角色弧的核心。
