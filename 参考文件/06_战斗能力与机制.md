# 流萤 — 战斗能力与机制

## 角色定位

流萤是**火属性 · 毁灭命途**的击破型输出角色。她的核心玩法围绕**终结技进入「完全燃烧」状态**，在此状态下大幅提升速度、击破效率和伤害输出。

## 基础属性（Lv.80）

| 属性 | 数值 |
|------|------|
| 生命值 | 较低（毁灭命途中偏脆） |
| 攻击力 | 中上 |
| 防御力 | 中等 |
| 速度 | 基础速度中等，完全燃烧状态下大幅提升 |
| 能量上限 | 较高 |

## 技能详解

### 普攻·指令-焚灭
- 对单体敌人造成火属性伤害
- 完全燃烧状态下强化为扩散攻击

### 战技·指令-灼燃
- 消耗自身生命值，对单体敌人造成火属性伤害
- 完全燃烧状态下强化为扩散攻击，伤害和削韧大幅提升

### 终结技·完全燃烧
- **核心机制**：进入「完全燃烧」状态
- 行动提前100%（立即行动）
- 速度大幅提升（+60）
- 获得强化普攻和强化战技
- 击破效率提升50%
- 效果抵抗提升
- 火焰变为翠绿色，面罩变形

### 天赋·茧式源火中枢
- 生命值越低，受到的伤害越低
- 完全燃烧状态下维持最大减伤
- 效果抵抗提升

### 秘技·焦土陨击
- 跳跃至空中自由移动
- 下落攻击大范围敌人
- 进入战斗时为全体敌人添加火属性弱点

## 额外能力（行迹）

### 偏时迸发
- 可对非火属性弱点的敌人造成削韧（击破效率降低但不会为零）

### 灼热甲壳
- 完全燃烧状态下，削弱受到的伤害

### 熔火爆裂
- 击破特攻按比例转化为完全燃烧状态下的伤害加成

## 星魂（命座）详解

### 第一星魂：荧火初燃
- 强化普攻和强化战技对弱点击破状态下的敌人造成额外伤害

### 第二星魂：残火复明
- 完全燃烧状态下，击破敌人弱点后获得额外回合

### 第三星魂：薪火相传
- 终结技等级+2，天赋等级+2

### 第四星魂：蝶火翩跹
- 完全燃烧状态下，受到致命伤害时免疫死亡一次（冷却时间较长）

### 第五星魂：心火相映
- 战技等级+2，普攻等级+2

### 第六星魂：星火燎原
- 完全燃烧状态下，提高火属性抗性穿透和击破效率的极限值

## 配队思路

### 超击破队（核心玩法）
- **流萤**（主C）+ **阮·梅**（击破辅助）+ **同谐主**（超击破触发器）+ **加拉赫**（击破治疗/副C）
- 通过击破敌人弱点触发超击破反应，造成巨额伤害

### 传统毁伤队
- **流萤**（主C）+ **阮·梅/布洛妮娅**（辅助）+ **存护角色**（生存位）+ **自由位**

### 低配开荒队
- **流萤**（主C）+ **艾丝妲**（加速+火伤加成）+ **同谐主**（超击破）+ **娜塔莎/玲可**（治疗）

## 战斗技巧

1. **能量管理**：优先充能进入「完全燃烧」状态，这是流萤输出的核心窗口
2. **击破节奏**：在完全燃烧状态下集中施放强化战技，最大化击破收益
3. **生命管理**：流萤消耗生命值施放技能，注意保持血量在安全范围（天赋提供低血减伤）
4. **弱点施加**：秘技进场自动施加火弱点，配合「偏时迸发」可跨属性击破

## 角色强度评价

### 优点
- 火属性击破队核心输出，机制独特
- 完全燃烧状态下的极速行动和强化攻击
- 自带火弱点施加，跨属性泛用性强
- 星魂提升平滑，2魂是关键质变点

### 缺点
- 对击破队友依赖度较高（尤其是阮·梅）
- 无完全燃烧状态时输出大幅下降
- 生命消耗机制对续航有一定要求
- 对单对群兼顾但非顶尖

## 萨姆装甲技术细节

### 火萤Ⅳ型战略强袭装甲
- 代号：SAM（Strategic Assault Mech - Firefly Type IV）
- 制造商：格拉默共和国（已覆灭）
- 动力源：未知能量炉，完全燃烧时发生质变
- 攻击方式：能量汇聚于腕甲及拳套，肘关节喷射动能
- 飞行能力：完全燃烧状态下具备高速飞行能力
- 生命维持：装甲内部设有医疗舱，维持患有失熵症的流萤的生命体征
- 环境适应：可进入高辐射星系、高重力星球等极端环境

### 完全燃烧状态
- **外观变化**：火焰由橙红色变为翠绿色，面罩变形，装甲表面纹路亮起
- **能量质变**：动力炉能量输出达到极限
- **意志外化**：完全燃烧是流萤强大意志力的具现化——当她真正为自己而战时，这股力量就会显现

> 「只要我能一直燃烧下去，我所在的地方，就是烈火，而非灰烬。」
